\# Sistema de Cartão do RU - SQL Server



Projeto acadêmico desenvolvido para praticar modelagem de banco de dados relacional.



\## Funcionalidades



\- Cadastro de alunos

\- Cadastro de cartões

\- Registro de empréstimos

\- Relacionamento com chaves primárias e estrangeiras

\- Consultas utilizando JOIN



\## Tecnologias utilizadas



\- SQL Server
